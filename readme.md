## 一个简单的单词翻译 api

* 环境
    
    python3.6.5

    selenium version: 3.14.0

    flask version: 1.0.2


* api 地址：

* conf.ini
    一个配置文件

* flaskapi.py
    flask 框架下的一个简易 api

* requestapi.py
    api 测试脚本

* translate.py
    利用爬虫做的一个单词翻译机（目前只有单词）

* chromedriver
    ChromeDriver 2.41578700

需要第三方库：

    selenium

    flask

    beautifulsoup 

需要 chromedriver 启动器
    
    注意， chromedriver 的版本，需要和 Selenium chrome 对应
    
    
    
可以先运行 setup.py 测试一下
